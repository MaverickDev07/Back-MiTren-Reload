Recibir el monto inicial a pagar
/api/efectivo/monto se usa de esta manera en metodo POST :
{
  "amount": 50
}
este es la respuesta esperada si todo esta bien:
{
  "TotalMonedero": 45.6,
  "BilletesAceptados": [10, 20, 50]
}

Actualizar el monto pagado conforme el usuario inserta dinero
/api/efectivo/pagar se usa de esta manera en metodo POST:
{
  "amount": 50
}
Respuesta esperada:
Si el pago sigue en progreso, la respuesta será algo como esto:
json
{
  "message": "Pago en progreso",
  "totalPaid": 50
}
Si el monto total pagado ya ha alcanzado el monto objetivo, la respuesta será algo como esto:

json
{
  "message": "Pago completado",
  "totalPaid": 150,
  "change": 0
}
